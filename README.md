### Project Description
The Love Chatbot is a fun project that simulates chatting with a virtual lover, giving sweet and affectionate replies. 💖
It creates an engaging experience with real-time messages, emojis, and a playful personality. 💬
Designed for entertainment, it blends creativity with simple chatbot technology. 🌸

### The Problem (that doesn't exist)
The problem we’re solving is that sometimes people just need a cute, caring chat buddy when no one is around. 

### The Solution (that nobody asked for)
Our solution is a virtual lover chatbot that always replies with sweet, romantic, and playful messages. 💖
We built a chat system that feels like texting your crush—full of emojis, affection, and fun vibes. 😍
In short, we created a 24/7 love machine that never ghosts you! 💬✨

# Installation
1. Clone the Project
Open your terminal and run:
git clone https://github.com/your-username/lovi-bot.git
cd lovi-bot
 2. Install Required Python Packages
Install all required dependencies listed in requirements.txt:
pip install -r requirements.txt
Note: Make sure Python and pip are already installed on your system.
 3. Configure API Key
Create a .env file in the root folder.
Add your Gemini or OpenAI API key:
GEMINI_API_KEY=your-api-key-here
